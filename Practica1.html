HTML

HTML.,siglas en ingés de HyperText Markup Lenguage hace referencia al lenguaje de marcado para la
elaboración de paginas web, no confundir con un lenguaje de programación. Es un estandar a cargo del World Wide Web
Consortium (W3C)
organización dedicada a la estandarización de casitodas las tecnologías ligadas a la web.

Primeras especificaciones de HTML

Timb Bernes-Lee1en 1991 describe 18 elementos que incluyen el diseño inicial y relativamente simple de HTML.
Bernes-Lee consideraba el HTML una ampliación de SGML, pero no fue formalmente reconocida como tal hasta la publicación a
mediados de 1993, por la IETF2, de una primera proposición para una especificación del HTML.:el borrador del HyperText
Markup Lenguage de Bernes-Lee y Dan Connolly.

Marcador HTML

El HTML se escribe en forma de < etiquetas >, rodeadas por corchetes angulares (< > /). El HTML tambien puede describir hasta
cierto punto la aparciencia de un documento y puede incluir o hacer referencia a un tipo de programa llamado script, el cual puede
afectar el comportamiento de navegadores web.
